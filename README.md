# 🚗 Laju-Nanggroe

Aplikasi Transportasi Online Sederhana Berbasis Web

Laju-Nanggroe adalah aplikasi pemesanan transportasi online berbasis
HTML, CSS, dan JavaScript (tanpa backend). Website ini dirancang dengan
tampilan modern, responsif, dan mudah digunakan di semua perangkat.

------------------------------------------------------------------------

## ✨ Fitur Utama

### 🏠 Beranda

-   Input tujuan perjalanan
-   Perhitungan tarif otomatis
-   Pilihan driver
-   Tombol pemesanan instan

### 💰 Sistem Saldo

-   Saldo pengguna (simulasi)
-   Top up saldo melalui modal popup
-   Update saldo secara real-time

### 🚗 Pemesanan

-   Validasi tujuan & saldo
-   Pilih driver
-   Setelah pesan:
    -   Saldo berkurang
    -   Data masuk ke riwayat
    -   Map muncul

### 🗺️ Map (Leaflet.js)

-   Menampilkan lokasi driver & tujuan
-   Simulasi perjalanan
-   Tombol Chat WhatsApp & Selesai

### 📜 Riwayat Pesanan

-   Menyimpan:
    -   Tujuan
    -   Driver
    -   Tarif
    -   Tanggal

### 👤 Profil Pengguna

-   Edit nama & nomor WhatsApp
-   Foto profil dari file lokal: image/foto.jpg
